# Human Activity Recognition using Machine Learning & Neural Networks

Classifying Walk vs Run from wearable sensor data using ML and Neural Networks.

## Dataset
- 88,588 rows | 11 features | Binary Classification
- Source: Run or Walk Reduced Dataset (Kaggle)

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- MLP Neural Network ← Best Model (99.24% accuracy)

## Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Results
| Model | Accuracy |
|-------|----------|
| MLP Neural Network | 99.24% |
| KNN Tuned | 99.16% |
| Random Forest Tuned | 99.01% |
| Decision Tree | 98.25% |
| Logistic Regression | 95.33% |
